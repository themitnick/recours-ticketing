🎯 Objectif du Test

Vérifier que la section "Décision Superviseur" s'affiche correctement quand :

  • ✅ L'utilisateur a le rôle SUPERVISEUR_RECOURS, SUPERVISEUR_IT ou ADMIN
  • ✅ Le statut de la demande est "EN_COURS", "NOUVEAU" ou "EN_ATTENTE_VALIDATION"
  • ✅ Les permissions canValidateNiveau1 ou canValidateNiveau2 sont activées

🔧 Boutons de Debug Disponibles

  • 👥 Superviseur Recours - Simule le rôle SUPERVISEUR_RECOURS
  • 💻 Superviseur IT - Simule le rôle SUPERVISEUR_IT
  • 🔧 Admin - Simule le rôle ADMIN
  • 🔄 EN_COURS - Force le statut à EN_COURS
  • 🆕 Test Data - Recrée des données de test
